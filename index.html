About This Website

This is a database of all the Generation 1 Pokémon. It covers the original 151 Pokémon found in the Kanto region.

You can search by name or number, view their images, and learn cool facts!

Page Overview

This website is split into 3 main pages:

1. The Home Page introduces the project.

2. The Pokédex Page displays all Pokémon with images and stats along with a search feature to find Pokémon quickly.

3. The Meet The Team Page lets you learn more about the amazing team behind this project!
